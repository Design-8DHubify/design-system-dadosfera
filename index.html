Design System Dadosfera

Fundamentos, 25+ componentes, um construtor plug & play para RD Station e templates de e-mail e pop-up. HTML, CSS e JavaScript nativos, sem dependências externas.

CSS Base — para todas as LPs

Como usar no RD Station

  1. No RD Station, edite a LP e adicione um bloco de HTML / Código (ou o CSS global da página).
  2. Cole as fontes da marca no <head>:
    <link rel="stylesheet" href="https://use.typekit.net/lrl6igp.css">
  3. Cole o CSS Base abaixo dentro de uma tag <style>…</style>.
  4. Pronto: use as classes .ds-btn, .ds-card, .ds-hero etc. em qualquer bloco — ou gere componentes prontos no RD Builder.
css/dadosfera-base.css
/* Carregando dadosfera-base.css… */
